# Logo Assets

Place your logo file here (e.g., `logo.png`, `logo.svg`, `logo.webp`).

Recommended:
- **Format**: SVG (scalable) or PNG (with transparent background)
- **Size**: 200x200px or larger (will be scaled down)
- **Name**: `logo.svg` or `logo.png`

After adding your logo file, it will automatically appear in:
- Login/Register pages
- Dashboard header
- Settings page
